# Python Programming Language Practice Exercises (CS1101)

A collection of Python programming exercises, solutions and practice programs developed as part of the **CS1101 – Fundamentals of Programming** course.

## 📖 About

This repository contains Python programs created while practicing fundamental programming concepts. The exercises cover a variety of topics ranging from basic syntax to problem-solving techniques and algorithm development.

The purpose of this repository is to:

* Strengthen Python programming skills
* Practice problem-solving techniques
* Understand fundamental programming concepts
* Maintain a record of coursework and self-learning activities

## 🚀 Topics Covered

* Python Basics
* Variables and Data Types
* Input and Output Operations
* Conditional Statements
* Loops
* Functions
* Lists and Tuples
* Dictionaries and Sets
* String Manipulation
* File Handling
* Exception Handling
* Problem Solving Exercises

## 🛠️ Requirements

* Python 3.x
* Any Python IDE or Code Editor

  * Visual Studio Code
  * PyCharm

## ▶️ Running the Programs

1. Clone the repository:

```bash
git clone https://github.com/Nipun-Lakshan/python-programming-language-practice-exercises-cs1101.git
```

2. Navigate to the project directory:

```bash
cd python-programming-language-practice-exercises-cs1101
```

3. Run a Python file:

```bash
python filename.py
```
